공항 분실물 플랫폼

인천·김포공항에 두고 온 물건,
다시 당신 손으로.

kustody는 인천국제공항(ICN)과 김포국제공항(GMP)의 유실물 회수·배송 서비스입니다. 온라인으로 신고하면 나머지는 저희가 처리합니다.

여행자와 공항을 연결하는
3단계 프로세스

전화도, 다시 날아갈 필요도 없습니다. 스마트폰으로 분실물을 찾으세요.

01

분실물 신고하기

온라인 양식으로 공항, 항공편 번호, 물품 정보를 제출하세요 — 24시간 언제든 가능합니다. 전화할 필요 없습니다.

02

모든 센터에 대신 연락합니다

kustody가 해당 공항의 모든 분실물 센터 — 항공사 데스크, 보안 검색대, 터미널 사무실 — 에 이메일로 동시에 연락합니다.

03

주소 입력 후 수령

배송비를 결제하고 주소를 확인하면 전 세계 어디든 배송해 드립니다.

// 기능

해결했습니다.

한 번의 신청으로 모든 센터에

공항에는 항공사 데스크, 보안 검색대, 터미널 사무실 등 여러 분실물 센터가 있습니다. 저희가 모든 곳에 동시에 연락하므로 직접 할 필요가 없습니다.

24시간 온라인 접수

전화도, 대기 음악도, 언어 장벽도 없습니다. 온라인 양식으로 언제든 접수하면 저희가 이메일로 대신 처리합니다.

글로벌 배송

국내 택배부터 국제 특급 배송(DHL, FedEx)까지 한 곳에서 처리합니다. 통관 서류도 대행합니다.

진행 상황 알림

접수 → 확인 → 픽업 → 배송 → 수령까지 단계별 업데이트. 직접 확인할 필요가 없습니다.

배송 보험

고가 물품은 배송 중 보험이 적용됩니다. 배송 중 분실이나 파손 시 빠른 보상을 제공합니다.

다국어 지원

영어, 한국어, 일본어, 중국어를 지원합니다. 해외 여행자도 언어 장벽 없이 플랫폼을 이용할 수 있습니다.
