# kustody.co

Static marketing site for [kustody.co](https://kustody.co).

## Quick Start

```bash
npm run dev
```

Opens a local server at `http://localhost:3000` serving the `public/` directory.

## Deploy

**Vercel** (recommended):

```bash
npm run deploy:vercel
```

**Netlify**:

```bash
npm run deploy:netlify
```
